# Meal Planner AI MCP

Generate weekly meal plans with macro targets and dietary preference support.

## Tools

- `plan_weekly_meals` — Generate a simple weekly meal plan.
- `calculate_daily_macros` — Sum macros from a list of foods.
- `suggest_substitute` — Suggest a substitute ingredient for dietary restrictions.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m meal-planner-ai-mcp.server
```
